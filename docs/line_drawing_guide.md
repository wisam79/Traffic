# دليل رسم الخطوط المتقدم - Advanced Line Drawing Guide

## 📋 نظرة عامة

تم تحسين نظام رسم الخطوط بالكامل ليدعم أوضاع متعددة، معاينة مباشرة، إدارة شاملة، والتراجع/إعادة.

## 🎯 أوضاع الرسم الثلاثة

### 1. **خط واحد (Single Line)**
- **الاستخدام**: رسم خط عد واحد
- **كيفية الاستخدام**:
  1. انقر لتحديد نقطة البداية (خضراء)
  2. انقر لتحديد نقطة النهاية (برتقالية)
  3. يظهر خط سماوي مع سهم الاتجاه
- **مناسب لـ**: تقاطعات بسيطة باتجاه واحد

### 2. **خطوط متعددة (Multi-Line)**
- **الاستخدام**: رسم عدة خطوط عد متتالية
- **كيفية الاستخدام**:
  1. انقر نقطة البداية
  2. انقر نقطة النهاية
  3. يتم حفظ الخط وتبدأ خط جديد تلقائياً
  4. كرر حسب الحاجة
- **مناسب لـ**: تقاطعات معقدة بعدة مسارات

### 3. **منطقة/مضلع (Zone/Polygon)**
- **الاستخدام**: رسم منطقة عد مغلقة
- **كيفية الاستخدام**:
  1. انقر لإضافة نقاط المضلع (3 نقاط على الأقل)
  2. انقر قرب نقطة البداية لإغلاق المضلع
  3. يظهر المضلع بحدود برتقالية وشفافية
- **مناسب لـ**: مناطق عد معقدة، مواقف السيارات

## 🎨 المعاينة المباشرة

أثناء الرسم، يظهر:
- **نقاط ملونة**: خضراء للبداية، برتقالية للباقي
- **خط أصفر مؤقت**: يمتد من آخر نقطة إلى موقع المؤشر الحالي
- **عداد النقاط**: يُظهر عدد النقاط المضافة

## 📏 إدارة الخطوط

### لوحة إدارة الخطوط

اضغط على **"📏 إدارة الخطوط"** في شريط العنوان لإظهار اللوحة التي تحتوي على:

#### **قائمة الخطوط**
- تعرض جميع الخطوط المرسومة
- يُظهر كل سطر: رقم الخط + الإحداثيات
- حدد أي خط لتحديده (سيُستخدم للحذف)

#### **أزرار التحكم**

| الزر | الوظيفة |
|------|---------|
| **↶ تراجع** | التراجع عن آخر عملية رسم |
| **↷ إعادة** | إعادة العملية المُتراجع عنها |
| **🗑 حذف المحدد** | حذف الخط المحدد من القائمة |
| **⊗ مسح الكل** | حذف جميع الخطوط |

#### **قائمة وضع الرسم**
- اختر الوضع: خط واحد، خطوط متعددة، أو منطقة
- تغيير الوضع يغير طريقة الرسم فوراً

## 🔄 التراجع والإعادة

### كيفية الاستخدام:
1. ارسم عدة خطوط
2. اضغط **"↶ تراجع"** - يتم حذف آخر خط
3. اضغط **"↷ إعادة"** - استرجاع الخط المحذوف

### المميزات:
- سجل تراجع غير محدود
- سجل إعادة منفصل
- يعمل عبر جميع أوضاع الرسم

## ✨ تحديد الخطوط

### طرق التحديد:
1. **من القائمة**: انقر على أي خط في قائمة الخطوط
2. **تحديد مرئي**: الخط المحدد يُبرز (قيد التطوير)

### بعد التحديد:
- اضغط **"🗑 حذف المحدد"** لحذفه
- يتم تحديث العداد تلقائياً

## 📊 التكامل مع AI Engine

### كيف يعمل:
1. عند اكتمال كل خط، يُرسل إلى AI Engine
2. أول خط في القائمة هو النشط حالياً للعد
3. يمكن تطويره ليدعم عدة خطوط عد متزامنة

## 💡 نصائح واحترافات

### ✅ أفضل الممارسات:
- ارسم الخطوط بوضوح عبر مسارات المركبات
- استخدم وضع "خطوط متعددة" للتقاطعات الكبيرة
- استخدم "منطقة" لمواقف السيارات
- استخدم التراجع بدلاً من مسح الكل عند الخطأ

### ⚠️ ملاحظات مهمة:
- الخطوط تُحفظ في الذاكرة فقط (تختفي عند إعادة التشغيل)
- يمكن رسم أي عدد من الخطوط
- كل خط له معرف فريد (ID)
- المضلع يحتاج 3 نقاط على الأقل

## 🎯 أمثلة استخدام

### مثال 1: تقاطع بسيط
```
1. اختر "خط واحد"
2. انقر عند بداية المسار
3. انقر عند نهاية المسار
4. ✅ تم! سيبدأ العد تلقائياً
```

### مثال 2: تقاطع متعدد المسارات
```
1. اختر "خطوط متعددة"
2. ارسم خط للمسار الأول
3. ارسم خط للمسار الثاني
4. كرر لكل مسار
5. ✅ جميع المسارات تُعد الآن!
```

### مثال 3: موقف سيارات
```
1. اختر "منطقة (مضلع)"
2. انقر عند كل ركن من أركان الموقف
3. انقر قرب النقطة الأولى لإغلاق المنطقة
4. ✅ المركبات داخل المنطقة تُعد!
```
